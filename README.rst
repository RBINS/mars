Description
===============
Mars is an application to manage your archeology items inside collections.
You can manage your content throught content types with some level of hierarchy:


    - Administration

        - Institutions

        - People

    - Repositories

    - Sites
    
        - Site Items

    - Collections
        
        - Collections Items
